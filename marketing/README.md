## Quick facts (kept consistent across every asset)
- **Tagline:** Quit the DMs. Get paid up front.
- **What it is:** A booking-and-deposit platform built for hair braiders.
- **Numbers:** 100% of deposits stay with the braider · 0% commission · ~15 min to set up · ~90s to book.
- **Stack:** Next.js 14 · TypeScript · Tailwind · Stripe (Connect + Elements) · Vercel.
- **Demo login:** any email/password; `amara@braidflow.app` opens the braider dashboard.
- **License:** Proprietary — All Rights Reserved. Available for acquisition/licensing.
